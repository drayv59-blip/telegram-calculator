# 🤖 Telegram Calculator Bot

Умный Telegram-бот для расчёта дозы инсулина с автоматическим определением коэффициента по времени суток.

## ✨ Возможности

- 💉 **Расчёт дозы инсулина** по сахару в крови и углеводам
- 🕐 **Автоматическое определение УК** (коэффициента) по времени суток
- ✍️ **Ручной ввод УК** при необходимости
- 💓 **Heartbeat** - периодические сигналы жизни бота
- 🔧 **Health Check** - проверка здоровья приложения
- ✅ **Круглосуточная работа** на облачном хостинге

## 🚀 Быстрый старт

### Локальный запуск

1. **Клонировать репозиторий:**
```bash
git clone https://github.com/drayv59-blip/telegram-calculator.git
cd telegram-calculator
```

2. **Создать виртуальное окружение:**
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. **Установить зависимости:**
```bash
pip install -r requirements.txt
```

4. **Создать файл `.env`:**
```bash
cp .env.example .env
```

5. **Заполнить `.env` с твоими данными:**
```bash
TELEGRAM_BOT_TOKEN=твой_токен_от_BotFather
ADMIN_ID=твой_ID_от_userinfobot
PORT=8000
```

6. **Запустить бота:**
```bash
python bot.py
```

### Деплой на Render.com (Рекомендуется)

1. **Создать аккаунт** на https://render.com

2. **Запушить репозиторий на GitHub:**
```bash
git add .
git commit -m "Initial commit"
git push origin main
```

3. **В Render:**
   - Нажать "New" → "Web Service"
   - Выбрать твой репозиторий
   - Build command: `pip install -r requirements.txt`
   - Start command: `python bot.py`

4. **Добавить переменные окружения:**
   - Environment Variables:
     - `TELEGRAM_BOT_TOKEN` = твой токен
     - `ADMIN_ID` = твой ID
     - `PORT` = 8000

5. **Нажать Deploy** → Бот будет работать 24/7! 🎉

### Деплой на Railway.app

1. **Создать аккаунт** на https://railway.app

2. **Подключить GitHub**

3. **Создать новый проект** из своего репозитория

4. **Добавить переменные окружения** в Variables

5. **Deploy** → Готово!

## 💉 Примеры использования

### Автоматический УК (по времени суток)
```
/c 9.3 72
```
- 9.3 - текущий уровень сахара (ммоль/л)
- 72 - количество углеводов (граммы)
- УК определяется автоматически по времени

### Ручной УК
```
/c 9.3 72 1.4
```
- 9.3 - уровень сахара
- 72 - углеводы
- 1.4 - твой коэффициент (УК)

### С запятой вместо точки
```
/c 9,3 72
```

## ⏰ Коэффициенты по времени суток

| Время | УК | Приём пищи |
|-------|-------|----------|
| 06:00 - 11:00 | 1.33 | Завтрак |
| 11:00 - 16:00 | 1.33 | Обед |
| 16:00 - 22:00 | 1.38 | Ужин |
| 22:00 - 06:00 | 1.30 | Ночь |

## 💓 Heartbeat (сигнал жизни)

Бот отправляет тебе сообщение каждый час, чтобы подтвердить, что он живой:

```
💓 HEARTBEAT - Бот живой!

🤖 Статус: ✅ Работает
⏰ Время: 2026-07-22 15:30:45
📊 Версия: 2.0 с Heartbeat
🌍 Зона: Europe/Moscow
```

Если heartbeat не приходит → сервис упал → можно перезагрузить.

## 🔧 Переменные окружения

```bash
TELEGRAM_BOT_TOKEN    # Токен бота от @BotFather
ADMIN_ID              # Твой Telegram ID для heartbeat
PORT                  # Порт для health-check (default: 8000)
```

## 📍 Как получить необходимые значения

### Получить TELEGRAM_BOT_TOKEN:
1. Открой Telegram
2. Найди бота `@BotFather`
3. Напиши `/newbot`
4. Следуй инструкциям
5. Скопируй токен

### Получить ADMIN_ID:
1. Найди бота `@userinfobot`
2. Напиши ему любое сообщение
3. Он вернёт твой ID

## 🐛 Решение проблем

### Бот не отвечает
1. Проверь, что `TELEGRAM_BOT_TOKEN` правильный
2. Посмотри логи: `python bot.py`
3. Убедись, что интернет подключен

### Heartbeat не приходит
1. Проверь, что `ADMIN_ID` установлен
2. Убедись, что это твой реальный ID
3. Посмотри в консоли логи ошибок

### Health Check не работает
1. Проверь, что PORT корректный (обычно 8000)
2. На Render PORT устанавливается автоматически

## 📚 Дополнительная информация

### Как работает Health Check
HTTP сервер отвечает "OK" на запросы с хостинга каждые 5 минут, чтобы сервис не "засыпал".
